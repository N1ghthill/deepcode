# deepcode

`deepcode` is the publishable CLI package for the DeepCode project.

- Repository: <https://github.com/N1ghthill/deepcode>
- Product documentation: <https://github.com/N1ghthill/deepcode#readme>
- Configuration reference: <https://github.com/N1ghthill/deepcode/blob/main/docs/16-configuration.md>

This package is built from the monorepo entrypoint in `apps/deepcode` and bundles the internal workspace packages required to run the terminal agent.
